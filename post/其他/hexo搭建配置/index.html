hexo搭建配置

Author: 王水泥 Date: 十一月 1, 2022  16:23:33 Category: 其他

初始化

  1. 新建文件夹/xx
  2. hexo init
  3. cd xx
  4. npm install

添加主题

网页：https://hexo.io/themes/

网页：https://hexo.io/plugins/

  1. 选择主题
  2. 找到其仓库地址
  3. cd themes
  4. git clone
  5. 修改_config.yml文件：theme:xx

本地服务器

  1. hexo g
  2. hexo server

其他

hexo clean 刷新配置

hexo new page tag
hexo new page category

_config.yml文件

创建文章

hexo new post “xx xx” or hexo new xx

注意：md文件的顶部需要加入 标记字符，否则不识别

上传

npm install hexo-deployer-git –save

hexo g -d
